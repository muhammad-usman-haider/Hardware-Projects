# Hardware Projects

A comprehensive collection of advanced Arduino and embedded systems projects demonstrating microcontroller programming, hardware integration, and real-world IoT applications. All projects are implemented in C++ for Arduino platforms.

## 📋 Overview

This repository contains professional-grade hardware projects that showcase embedded systems design, sensor integration, and real-time control systems. Each project is fully documented with hardware specifications, code examples, and troubleshooting guides.

## 🚀 Projects

### 1. [Arduino Password-Protected Door Lock System](./Arduino%20Password-Protected%20Door%20Lock%20System/)

A sophisticated access control system implementing secure door locking/unlocking through PIN-based authentication.

**Key Features:**
- 🔐 4-digit PIN authentication with keypad input
- 🔓 Servo motor-controlled electromagnetic lock
- 📱 16x2 LCD display with real-time feedback
- 🔊 Audio and visual alert system
- 🚨 Emergency manual override
- ⏱️ Automatic re-locking mechanism

**Technology Stack:** Arduino Uno, 4x3 Keypad, I2C LCD, Servo Motor

**Status:** ✅ Production Ready

---

### 2. [LED Control with Voice Commands](./LED%20Control%20with%20Voice%20Commands/)

An intelligent LED control system utilizing voice command recognition and serial communication for wireless device management.

**Key Features:**
- 🎙️ Voice command processing via serial communication
- 💡 Multi-LED simultaneous control (3x independent circuits)
- ⚡ Real-time response to user commands
- 🔄 Extensible architecture for additional LEDs
- 📊 Case-insensitive command matching

**Technology Stack:** Arduino (UNO/Mega), Serial Communication, GPIO Control

**Status:** ✅ Fully Tested

## 🔧 Getting Started

### Prerequisites

- Arduino IDE (v1.8.0 or higher) - [Download](https://www.arduino.cc/en/software)
- Arduino compatible microcontroller board
- USB cable for board programming
- Git for cloning the repository

### Installation

```bash
# Clone the repository
git clone https://github.com/muhammad-usman-haider/Hardware-Projects.git

# Navigate to your desired project
cd Hardware-Projects/[Project-Name]

# Follow project-specific README instructions
```
